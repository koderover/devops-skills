# 贡献你的技能！

## 为什么贡献？
- 让更多人享受你踩坑后的经验
- 建立个人影响力
- 丰富 DevOps 生态

## 如何贡献

### 方式一：提交 PR

1. Fork [https://github.com/koderover/devops-skills](https://github.com/koderover/devops-skills)
2. 创建分支：`git checkout -b skill/your-skill-name`
3. 在 `skills/` 目录下创建新技能
4. 提交 Pull Request

### 方式二：直接分享

联系 Maintainer，直接添加你的技能

## 技能模板

```markdown
# 技能名称

一句话描述。

## 环境变量

| 变量名 | 必填 | 说明 |
|--------|------|------|
| VAR | 是 | 说明 |

## 核心功能

1. 功能一
2. 功能二

## 使用示例

\`\`\`bash
# 示例
\`\`\`
```

## 审核标准

- 功能描述清晰
- 示例可直接使用
- 无明显错误

## 技能命名规范

- 使用小写字母
- 单词之间用连字符 `-` 分隔
- 前缀使用工具/平台名称
- 示例：`devops-jenkins-quickstart`、`devops-harbor-setup`
